# TEFA
TEFA - TimeBank Event Frame Annotation




# How to use

## Create a virtual environment
   ```bash
   $ python3 -m venv env
   $ ./env/bin/activate
   ```
## Install the packages.
   ```bash
   $ pip install -r requirements.txt
   ```
   
## Execute
   ```bash
   $ cd src/
   $ python3 mainpanel.py
   ```
